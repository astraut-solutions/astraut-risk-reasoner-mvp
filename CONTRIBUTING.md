# Contributing

Thanks for contributing to Astraut Risk Reasoner.

## Development Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
make install
```

3. Run checks:

```bash
make lint
make test
```

## Pull Requests

1. Create a branch from `main`.
2. Keep changes focused and include tests when behavior changes.
3. Update docs for user-facing changes.
4. Open a pull request with a clear summary and validation steps.

## Reporting Issues

Use GitHub issue templates for bugs, features, and security concerns.
